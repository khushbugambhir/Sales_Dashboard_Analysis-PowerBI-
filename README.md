Sales Dashboard Analysis (Power BI)

Overview :-

This project is an interactive Sales Dashboard designed to analyze business performance across multiple dimensions such as sales, profit, discounts, and geographic distribution. The dashboard provides actionable insights to support data-driven decision-making.

Key Objectives :-

1)Track overall business performance.

2)Identify sales patterns over time.

3)Analyze the impact of discounts on sales.

4)Understand geographic sales distribution.

5)Evaluate profitability against revenue.

Key Insights :-

1)Certain cities contribute significantly higher sales.

2)Promotional discounts vary across categories and influence sales.

3)Sales show an increasing trend over time with periodic spikes.

4)Strong correlation observed between net sales and profit.

Tools & Technologies :-

Power BI – Data visualization and dashboard creation

Excel / CSV – Data source and preprocessing
